RoR-Installation

schnell und einfach mit rvm unter Ubuntu

von Arndt Touby / @arndttouby

benötigte Pakete installieren

Shell öffnen und folgenden Befehl ausführen:


$ sudo aptitude install bash curl git patch build-essential openssl libreadline6 libreadline6-dev curl git-core zlib1g zlib1g-dev libssl-dev libyaml-dev libsqlite3-dev sqlite3 libxml2-dev libxslt-dev autoconf libc6-dev libgdbm-dev ncurses-dev automake libtool bison subversion pkg-config libffi-dev

rvm mit ruby und rails installieren

Shell öffnen und folgenden Befehl ausführen:


$ \curl -L https://get.rvm.io | bash -s stable --rails

Letzter Schritt

Shell reloaden


$ source ~/.rvm/scripts/rvm
